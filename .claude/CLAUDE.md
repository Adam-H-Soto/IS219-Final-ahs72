## Design Direction
- Vibe: Warm, collaborative, builder energy — approachable but technically serious
- Visual: Clean layout, warm neutrals (cream/amber) and deep teals, serif + mono typography
- Audience promise: "I build data tools that make the complex feel simple — now with AI"

## Tech Constraints
- Pure HTML, CSS, and vanilla JS — no frameworks, no build tools
- Must be deployable to GitHub Pages or Vercel with zero configuration
- All assets self-contained in a single index.html or split into index.html + style.css

## Development Approach
Spec-driven development: Requirements → Design → Tasks → Implementation.
Do not write any code until all three spec documents are approved.
